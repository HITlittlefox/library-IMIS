为什么那么多人喜欢吃猪头肉？ - Gucci Yang的回答 - 知乎

我第一次吃猪头肉，是在我很小的时候。

上个世纪九十年代末，外公因为做服装生意，有钱，天天给我买吃的，所以我从小跟在外公屁股后头混。

外公朋友众多，有开草纸厂的，有开小卖部的，有开船的，有做粮油加工的…但我最喜欢的，还是那个开熟食铺的朋友，好像姓王，也记不得了。老王六十来岁，身材瘦小，精神矍铄。

熟食铺开在村口，村口的南边有一条不大不小的河，老王常把剃下来的骨头反手扔进河中。这是条活河，常有货船往来，但不是鲁迅先生笔下的乌篷船，驶过村口时横七竖八躺了一岸，下船买点日用品，自然和村里人就熟络了。

“老王，你倒是扔两块肉下来啊！”

“肉？你还想吃肉？鱼都没有吃个屁肉！啃骨头吧！”

河的东边立着一根电线杆和几棵柳树，有几株叶片即将枯黄的藤蔓蔓延至电线杆腰身部位，死死缠住，从上面垂下来几个摇摇欲坠的丝瓜。而柳树之间，更是横牵了一道细细的黑绳，上头挂着几条盐焗过的大花鲢，这鱼都是停岸的船夫和老王用来置换猪头肉的，再有就是两片卤过还喷着浓重香料味的猪头。

猪头肉无需特别包装，要多少猪头肉，便切多少，不差毫厘。老王把肉切好，装进自带的大茶缸，或是拿黄土纸一包。猪头肉渗油多，薄薄一层纸怎么挡得住油脂的外溢，常手上都是油，怕浪费，赶忙伸出舌头一遍一遍舔净。

外公有个老朋友来做客，外婆的待客家宴便是半斤猪头肉炒荸荠。雪白的荸荠片经大火爆炒后变得脆嫩和爽滑，再加上红绿分明的各色辣椒，颜色分外灿烂。

还有半斤，在米饭上方支两根长短相同的木头，猪头肉置于碗中，随着米饭的蒸熟，肉也变得软烂。起锅时撒一把葱花，倒两滴香油，能把和尚馋得还俗。

我吃猪头肉，特别爱吃两个地方，一个是猪的眼睛部分，那是一块纯瘦肉，香味厚道；还有就是含在腮帮骨夹缝里的核桃肉，份量不多，酥糯爽口。

长大后，我读了一些大师写的文章，才知道有不少文豪也偏爱猪头肉：丰子恺先生在《缘缘堂随笔》中写他在杭州和老友吃猪头肉，“一斤猪头肉，每人照例是一斤老酒”。

周作人先生也作过一篇题为《猪头肉》的文章：“在摊上用几个钱买猪头肉，白切薄片，放在干荷叶上，微微洒点盐，空口吃也好，夹在烧饼里最是相宜，胜过北方的酱肘子。”

沈从文先生也极爱猪头肉，“下班后常在小手绢包点猪头肉回家，夹在冷馒头里吃，让你真是觉得亲近可爱。”

汪曾祺先生也爱吃猪头肉，他认为在夏日的黄昏，就着猪头肉喝二两酒，拎个马扎踅到一个荫凉树下纳凉，该是人生莫大的享受。
